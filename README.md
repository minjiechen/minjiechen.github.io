# Princeton Power Electronics

This repository contains the source code for the Princeton Power Electronics website (hosted at [minjiechen.github.io](https://minjiechen.github.io/)).

## Overview

The site is built with Jekyll, utilizing the Beautiful Jekyll theme.

## Local Development

To run this website locally for development and testing:

1. Ensure you have Ruby and Bundler installed.
2. Install the necessary dependencies:
   ```bash
   bundle install
   ```
3. Start the local development server:
   ```bash
   bundle exec jekyll serve
   ```
4. Open your web browser and navigate to `http://127.0.0.1:4000/`.
